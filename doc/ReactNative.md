### React Native

#### 运行环境参照官网

#### 基本命令
    ```
    npm start                   运行------->./rn目录下
    npm install --save xxxx     第三方库------->./rn目录下
    npm list                    查看依赖包
    ```
#### 准备
* es5，es6基本语法
* react基础
* android或ios基本知识


#### 功能

* 基本ui的处理
* Tab的使用
* 导航跳转的处理
* redux交互逻辑的处理
* 组件的编写
* 自定义widget
* 原生交互

#### 第三方库

* react-navigation
* redux
